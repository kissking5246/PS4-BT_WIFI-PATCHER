# PS4-BT_WIFI-PATCHER

PS4 BT_WIFI Patcher is a simple tool to repair Torus Firmware (Bt/Wifi related). 
This firmware known to be corrupted and causing the DS4 controller fail to connect 
to PS4 console wirelessly.
